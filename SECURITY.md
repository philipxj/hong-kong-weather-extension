# Security Policy

## Reporting a Vulnerability

Please do not open a public GitHub issue for security reports.

Email security concerns to:

```text
hello@fireshark.tech
```

Include:

- A clear description of the issue
- Steps to reproduce it
- Affected browser and extension version
- Any relevant screenshots, logs, or proof of concept details

We will review security reports as soon as possible and coordinate a fix before public disclosure when appropriate.

## Supported Versions

The latest published version of Hong Kong Weather is the supported version.
